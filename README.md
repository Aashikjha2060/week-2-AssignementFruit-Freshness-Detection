# Fruit Freshness Dataset EDA

This workspace contains a beginner-friendly Jupyter Notebook for a Neural Networks assignment.

## Main File
- [FruitFreshness_EDA.ipynb](FruitFreshness_EDA.ipynb)

## Dataset
The notebook automatically detects the dataset inside the workspace:
- Dataset root: `Data/fruits-360_100x100/fruits-360`
- Splits: `Training` and `Test`
- Number of classes: `260`
- Total images: `182,945`
- Image format: `JPG`
- Image size: `100x100`

## What the Notebook Includes
- Dataset overview
- Exploratory Data Analysis (EDA)
- Class counts and summary tables
- Bar chart, pie chart, sample images, and image size distribution
- Basic preprocessing steps:
  - resize
  - normalize
  - label encoding
- Short conclusion for the next neural network assignment

## Note
No model is trained in this notebook. It only prepares the dataset and documents the findings needed for the assignment report.
